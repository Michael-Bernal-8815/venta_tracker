2. Descripción del Servicio

Nuestra aplicación emplea tres modelos de Redes Neuronales Convolucionales (CNN) para mejorar la experiencia del usuario:

  • Reconocimiento de Texto: Este modelo está diseñado para identificar y analizar texto en imágenes.
  • Reconocimiento de Lugares: Este modelo identifica y clasifica ubicaciones geográficas o puntos de interés en las imágenes.
  • Reconocimiento de Objetos: Este modelo detecta y clasifica objetos en imágenes.
